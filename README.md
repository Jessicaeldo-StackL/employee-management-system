# Employee Management System

## Overview

Employee Management System developed using Nuxt.js, Rust (Actix Web), and PostgreSQL.

The application provides role-based authentication for Admin and Employee users, employee management functionality, file upload support, and leave management workflows.

## Technology Stack

### Frontend

* Nuxt.js

### Backend

* Rust (Actix Web)

### Database

* PostgreSQL

## Features

### Authentication

* JWT Authentication
* Role-Based Login (Admin / Employee)
* Protected Routes
* Session Management

### Employee Management

* Create Employee
* View Employee List
* View Employee Details
* Edit Employee
* Delete Employee
* Employee Profile Management

### File Management

* ID Proof 1 Upload (PDF)
* ID Proof 2 Upload (PDF)
* Profile Image Upload
* File View and Download

### Leave Management

* Apply Leave
* View Leave History
* Approve Leave Requests
* Reject Leave Requests
* Withdraw Leave Requests
* Admin Leave Monitoring

## Database

* PostgreSQL Database
* Database Migrations
* Structured Schema Design
